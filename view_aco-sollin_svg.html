Langkah-langkah b-ACO Sollin/Borůvka untuk d-MST

  1. Mulai.
  2. Input: graf berbobot, jumlah simpul n, batas derajat d, parameter ACO α, β, ρ, Q, jumlah semut, dan iterasi maksimum.
  3. Inisialisasi feromon: set nilai awal feromon pada semua edge, misalnya τ₀.
  4. Untuk setiap iterasi dari t = 1 sampai MaxIter:
    1. Set solusi terbaik iterasi = kosong.
    2. Untuk setiap semut:
      1. Inisialisasi komponen: setiap simpul dianggap 1 komponen, solusi edge E = ∅, derajat setiap simpul = 0.
      2. Ulangi selama jumlah komponen > 1:
        1. Untuk setiap komponen, buat daftar kandidat edge yang keluar dari komponen menuju komponen lain.
        2. Pilih satu edge menggunakan probabilitas ACO berdasarkan feromon dan heuristik biaya, dengan syarat:
          • tidak membentuk siklus antar komponen,
          • tidak melanggar batas derajat: derajat u + 1 ≤ d dan derajat v + 1 ≤ d.
        3. Kumpulkan edge terpilih dari semua komponen.
        4. Tambahkan edge-edge yang valid ke solusi; jika terjadi konflik atau duplikasi, ambil yang tetap valid.
        5. Update komponen: gabungkan komponen yang tersambung oleh edge-edge tersebut; update derajat simpul.
      3. Hitung cost solusi semut, yaitu total bobot edge pada solusi.
      4. Simpan jika solusi semut lebih baik dari solusi terbaik iterasi.
  5. Update feromon:
    • Evaporasi: kurangi feromon semua edge dengan faktor ρ.
    • Deposit: tambahkan feromon pada edge solusi terbaik, misalnya proporsional terhadap 1 / cost.
  6. Cek terminasi (opsional early stopping): jika solusi terbaik tidak membaik selama beberapa iterasi, hentikan lebih awal.
  7. Output: solusi terbaik keseluruhan berupa tree yang memenuhi batas derajat.
  8. Selesai.
Catatan singkat: “paralel tingkat komponen” berarti pada setiap fase Borůvka, setiap komponen dapat memilih edge keluar secara independen sebelum proses merge dilakukan.
